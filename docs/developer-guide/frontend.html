⚛ Frontend Architecture

Tech Stack

  • Framework: Next.js 16 (App Router)
  • Language: TypeScript (strict mode)
  • Styling: Tailwind CSS v4 + shadcn/ui
  • Wallet: wagmi v2 + viem
  • Charts: Custom SVG candlestick charts + CoinGecko API
  • Icons: Lucide React
  • State: React Query (@tanstack/react-query)

Project Structure

State Management

  • Wallet: wagmi hooks (useAccount, useBalance, useChainId)
  • Contract reads: useReadContract, useReadContracts (wagmi)
  • Contract writes: useWriteContract + useWaitForTransactionReceipt
  • Caching: @tanstack/react-query (configured with 10s stale time)
  • Local state: React useState for form inputs and UI state

Component Patterns

  • Server/Client split: Pages are server components; interactive UI is in client components
  • Animated sections: AnimatedSection wraps content with Intersection Observer fade-in
  • Error handling: Every component handles loading, error, empty, and not-deployed states
  • Accessibility: Skip-to-content link, aria-labels, focus-visible styles, reduced-motion support
