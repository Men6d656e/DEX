🚀 Deployment Guide

Prerequisites

  • Foundry (forge, cast, anvil)
  • Node.js >= 18.0.0
  • Git

1. Install Dependencies

git clone https://github.com/Men6d656e/DEX.git
cd DEX
make install

2. Build Contracts

make build-contracts

3. Run Tests

# All tests
make test

# Just contract tests
make test-contracts

# Just frontend tests
make test-frontend

# Coverage report
make coverage

4. Local Development (Anvil)

Start a local Anvil node in one terminal:

make anvil

In another terminal, deploy contracts:

make deploy-anvil

You'll be prompted to enter a private key interactively. For Anvil, you can use one of the default private keys printed when Anvil starts.

Update Contract Addresses

After deployment, the script outputs contract addresses in JSON format. Copy these into frontend/src/lib/constants.ts.

5. Start Frontend

make install-frontend
make build-frontend

# Or for development with hot reload:
cd frontend && npm run dev

The frontend will be available at http://localhost:3000.

6. Sepolia Testnet

Set up environment variables:

export SEPOLIA_RPC_URL=https://rpc.sepolia.org

Deploy:

make deploy-sepolia

You'll be prompted for a private key interactively (use a wallet with Sepolia ETH for gas).

7. Production Build

make build

The frontend build is output to frontend/.next/ and can be deployed to any Node.js hosting platform (Vercel, Railway, etc.).

8. CoinGecko API Key (Optional)

To use live market data instead of simulated charts:

  1. Get a free API key from CoinGecko
  2. Set it in your environment: NEXT_PUBLIC_COINGECKO_API_KEY=your_key_here
  3. For local development, add it to frontend/.env.local
