What is the DEX Dashboard?

The DEX Dashboard is an educational Web3 application that simulates a decentralized exchange (DEX) environment. It allows you to:

  • Claim mock tokens — Get free mETH and mBTC tokens from the faucet (once per day)
  • Swap tokens — Trade between mETH and mUSDC with transparent pricing
  • Track markets — View real-time charts and analytics powered by CoinGecko
