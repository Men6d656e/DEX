🚀 Getting Started

1. Prerequisites

  • A modern web browser (Chrome, Firefox, Brave, or Edge)
  • MetaMask browser extension installed
  • For local development: Anvil running on http://127.0.0.1:8545

2. Connect Your Wallet

  1. Open the DEX Dashboard in your browser.
  2. Click the "Connect Wallet" button in the top-right corner.
  3. MetaMask will prompt you to select an account and connect.
  4. Once connected, the button will show your network name and truncated address (e.g., ANVIL 0x1234...5678).
Tip: If you're using a local Anvil node, make sure to switch MetaMask to the Anvil network (Chain ID: 31337) or let MetaMask auto-detect it.

3. Navigate the Dashboard

The top navigation bar provides access to all major sections:

  • Home — Landing page with feature overview and stats
  • Faucet — Claim mock tokens (mETH, mBTC)
  • Swap — Trade tokens (mETH ↔ mUSDC)
  • Analytics — Charts, token stats, and portfolio tracking

4. Next Steps

  • Claim tokens from the Faucet
  • Execute a swap on the DEX
  • Explore the Analytics dashboard
